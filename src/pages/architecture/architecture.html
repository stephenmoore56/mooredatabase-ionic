Mobile Architecture Front End

The mobile front end of this application was constructed with:

  • Ionic Framework
  • Google Maps API
  • Plotly.js
 Back End

The back end of this application is a JSON API hosted by AWS on EC2 instances running Ubuntu 16.04 LTS / Apache 2.4. The API was constructed with:

  • PHP 7
  • Lumen 5.2
  • mongodb package
  • MongoDB 3.2.12
  • MongoDB Aggregation Pipelines
  • Gulp
  • Jasmine
  • Frisby
 Development Tools

For development, I use:

  • PHPStorm
  • WebStorm
  • Sublime Text
  • MongoDB Compass
  • git / GitHub
  • Ionic View
